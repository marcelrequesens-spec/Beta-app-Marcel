## 🎮 Cómo Empezar

### Opción 1: Ejecución Local (Recomendado)
```bash
# En tu terminal, navega a la carpeta del proyecto
cd ruta/a/trading_casino_pro

# Ejecuta el servidor
node server.js

# O simplemente:
bash START.sh
```

Luego abre tu navegador en: **http://localhost:3000**

### Opción 2: Desde tu Red Local
Si quieres acceder desde tu móvil o desde otro dispositivo en la misma WiFi:

1. Ejecuta el servidor (paso anterior)
2. Obtén tu IP local:
   - **Mac/Linux**: `hostname -I`
   - **Windows**: `ipconfig` (busca IPv4 Address)
3. En tu móvil/otro dispositivo: `http://TU_IP:3000`

Ejemplo: `http://192.168.1.100:3000`

### Opción 3: Línea de Comandos (Una línea)
```bash
cd trading_casino_pro && node server.js
```
